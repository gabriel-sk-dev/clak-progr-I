# Aula 03 - Exercício 01
## Usando while e if
Criar um aplicativo que respeite os seguintes requisitos:
-> Exigir um login (usuário e senha)
-> Ao informar o usuário "joao" e senha "123456" deve apresentar uma tela de boas vindas com o nome do usuário
-> Usuário ou senha errada deve ser exibida mensagem de erro e solicitar o login novamente
